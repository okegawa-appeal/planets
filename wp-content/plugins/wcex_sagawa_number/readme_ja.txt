******************************************
　　　佐川急便e飛伝Ⅲ　連携プラグイン
　　　WCEX Sagawa Number
******************************************


WCEX Sagawa Number は Welcart 専用の拡張プラグインで、日本国内のみの利用を想定しています。

佐川急便の送り状発行サポートシステム「e飛伝Ⅲ」との連携プラグインです。
送り状の一括発行作業や、お客様への荷物追跡番号の案内を効率化します。

ご利用方法の詳細は、導入マニュアルをご覧ください。
https://welcart.com/docs/SagawaNumber/WCEX_SagawaNumber.pdf


==============
ご利用方法
==============

【インストール】
管理パネルのプラグインの新規追加画面（プラグインのインストール）から
「アップロード」リンクをクリックして、ダウンロードした WCEX Sagawa Number の
圧縮ファイル（[例]wcex_sagawa_number.2.0.0.zip）をそのままアップロードして
有効化します。


【準備】
このプラグインをご利用になるには、佐川急便の送り状発行サポートシステム「e飛伝Ⅲ」を
ご利用いただいていることが前提となります。
